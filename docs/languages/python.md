## Data Types

The base data types map to Python accordingly:

| Protocol Type | Python type |
|---------------|-------------|
| `byte`        | `int`       |
| `bool`        | `bool`      |
| `int16`       | `int`       |
| `uint16`      | `int`       |
| `int32`       | `int`       |
| `uint32`      | `int`       |
| `int64`       | `int`       |
| `uint64`      | `int`       |
| `float`       | `float`     |
| `double`      | `float`     |
| `string`      | `str`       |
